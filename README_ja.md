# 短歌画像メーカー

ブラウザ上で短歌を縦書き画像として作成し、PNGで保存できるツールです。

## 概要
短歌画像ジェネレーターは、入力した短歌をHTML Canvasに描画し、画像として書き出します。ビルドやサーバーは不要で、`index.html` を開くだけで利用できます。

## 主な機能
- 日本語短歌向けの縦書き描画
- 入力と同時にプレビュー更新
- フォント選択（Google Fonts）
- 文字色・背景色の変更
- 設定タブ切り替え:
  - 短歌の設定
  - 作者名の設定
- フッター情報の表示:
  - 作者名
  - SNSアカウント（X / Instagram / Threads）
  - ラベル表示ON/OFF（項目ごと）
- レイアウトプリセット:
  - 複数行向け
  - 一行向け
- 詳細モードでの文字設定入力
- 出力サイズ切り替え:
  - 800 x 800
  - 960 x 1280
- ワンクリックPNG保存（`my-tanka.png`）

## 使い方
1. このリポジトリを取得します。
2. ブラウザで `index.html` を開きます。
3. 短歌とデザイン設定を調整します。
4. 「画像を保存する」をクリックして保存します。

追加のインストールやビルド作業は不要です。

## 利用時のメモ
- 縦書きの見た目を安定させるため、句ごとに改行して入力するのがおすすめです。
- 入力内容や設定を変更すると、キャンバスが即時再描画されます。
- フッター情報は画像下部に描画され、長い場合は自動で折り返されます。
- 現在の実装では、詳細設定値はプリセットと同期しつつ、短歌本文は横方向で常に中央配置されます。
- 詳細設定では上余白を調整でき、短歌本文の開始Y位置に反映されます。

## ファイル構成
- `index.html` - 画面構成
- `style.css` - スタイル定義
- `script.js` - ES Modulesのエントリーポイント
- `js/app.js` - 初期化フロー
- `js/tanka-renderer.js` - 短歌描画のオーケストレーション
- `js/dom.js` - DOM参照の収集とキャッシュ
- `js/state.js` - UI実行時状態
- `js/layout-settings.js` - レイアウト計算とUI同期
- `js/font-adjustments.js` - フォント/文字ごとの補正設定
- `js/text-layout.js` - 縦書きグリフ配置と折り返し計算
- `js/canvas-renderer.js` - Canvas描画（本文/フッター）
- `js/svg-renderer.js` - SVG描画とフォールバック処理
- `js/ui-events.js` - イベント登録の集約
- `README.md` - 英語版ドキュメント
- `README_ja.md` - 日本語版ドキュメント

## 設計メモ
- 機能責務ごとにモジュールを分割し、`dom` と `state` を明示的に受け渡す構成にしています。
- `script.js` は `initApp()` の呼び出しのみを担当します。
- 初回描画タイミングは `document.fonts.ready` で統一し、従来の初期化順序を維持しています。
- 標準はSVG描画で、失敗時はCanvasグリフ描画にフォールバックします。
