# 短歌画像メーカー - 確認用テスト文

## テスト1: 基本的な短歌
```
ひさかたの
光のどけき
春の日に
しづ心なく
花の散るらむ
```

## テスト2: 句読点を含む短歌
```
春来たり
野辺に咲き出
桜かな
白く香りて
見ゆるかぎりに
```

## テスト3: 括弧と長音を含む短歌
```
「春」という
字書きて思ふ
そは我が
心の一文字
―人生の如し
```

## テスト4: 様々な記号を含む短歌
```
月・星・そして
太陽までも
輝きて…
夜明け待つ身の
心ときめく
```

## テスト5: フル記号テスト（複合要素）
```
【異国へ】
ー海を越えて
行くとせば
心は常に
「故郷」想う也
```

---

### テスト方法
1. 上記の短歌テキストをコピーして、アプリケーションの「短歌を入力」欄に貼り付ける
2. フォント・背景色・文字色を変更して、各要素の配置を確認する
3. 以下をチェック：
   - ✓ 長音「ー」が90度回転している
   - ✓ 括弧「「」」「【】」が90度回転している
   - ✓ 句読点「、。」が右上にオフセットされている
   - ✓ 通常の文字が等間隔で縦に配置されている
