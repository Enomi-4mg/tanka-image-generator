## 回帰チェックシート（移行後）

### A. 基本描画
- [ ] 初回表示で短歌が表示される
- [ ] テキスト入力時に即時再描画される
- [ ] フォント切替で表示が変わる
- [ ] 文字色・背景色が即時反映される

### B. 方式切替
- [ ] 描画方式を「標準（SVG）」にすると描画される
- [ ] 描画方式を「簡易（Canvas）」にすると描画される
- [ ] 方式を連続切替しても描画が消えない

### C. レイアウト設定
- [ ] プリセット切替で文字サイズ/列間/上余白/文字間が更新される
- [ ] 詳細設定で文字サイズ変更が描画に反映される
- [ ] 詳細設定で列間変更が描画に反映される
- [ ] 詳細設定で上余白変更が本文の開始Y位置に反映される
- [ ] 本文が横方向で自動的に中央配置される

### D. フッター
- [ ] 作者名のみ入力で下部に表示される
- [ ] SNSのON/OFFで表示が切り替わる
- [ ] ラベルON/OFFが表示に反映される
- [ ] フッターが長文でも折り返される

### E. 出力
- [ ] 800x800 に切り替えるとヒント表示が正方形文言になる
- [ ] 960x1280 に切り替えるとヒント表示が縦長文言になる
- [ ] 画像保存でPNGが出力される
